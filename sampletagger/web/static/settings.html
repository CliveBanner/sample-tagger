Library

Stage 1 — Discover

Walk the filesystem, register new files, mark deleted ones. Extracts path hints for free — no audio decoding.

Skips a stat() call per file. Safe and much faster on network/cloud mounts where stat() is slow. Only new files are fully checked.
The scanner caches the file list to avoid slow pcloud walks. Enable this once after adding new files, then turn it off again.

Stage 2 — Classifiers & Embeddings

Compute classifier results and embeddings for indexed files. Each model writes to its own column or index — results are independent and never overwrite each other automatically.

Parallel processes. PANNs loads ~1 GB of model weights per worker.
Only files missing that result are processed. Already-labeled files are skipped unless Redo is set.
